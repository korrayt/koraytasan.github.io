## 🚀 GitHub'a Yükleme Talimatları

### Ön Gereksinimler

1. **Git kurulu olmalı** - [git-scm.com](https://git-scm.com/download/win) adresinden indirebilirsiniz
2. **GitHub hesabınız olmalı** - [github.com](https://github.com) adresinden oluşturabilirsiniz

### Adım 1: GitHub'da Yeni Repository Oluşturun

1. [github.com/new](https://github.com/new) adresine gidin
2. **Repository name:** `koraytasancom` yazın
3. **Public** seçeneğini işaretleyin
4. "Create repository" butonuna tıklayın

### Adım 2: Terminalde Komutları Çalıştırın

PowerShell veya Terminal'i açın ve şu komutları sırayla çalıştırın:

```powershell
# Proje klasörüne gidin
cd e:\koraytasancom

# Git'i başlatın
git init

# Tüm dosyaları ekleyin
git add .

# İlk commit'i yapın
git commit -m "Portfolio sitesi - ilk yükleme"

# Ana branch'i ayarlayın
git branch -M main

# GitHub repository'nizi bağlayın (KULLANICI_ADI kısmını kendi GitHub kullanıcı adınızla değiştirin)
git remote add origin https://github.com/KULLANICI_ADI/koraytasancom.git

# Dosyaları GitHub'a gönderin
git push -u origin main
```

### Adım 3: GitHub Pages'i Aktifleştirin

1. GitHub'daki repository sayfanıza gidin
2. **Settings** (Ayarlar) sekmesine tıklayın
3. Sol menüden **Pages** seçeneğine tıklayın
4. **Source** bölümünde:
   - Branch: `main` seçin
   - Folder: `/ (root)` seçin
5. **Save** butonuna tıklayın

### Adım 4: Sitenizi Görüntüleyin

Birkaç dakika bekledikten sonra siteniz şu adreste yayınlanacak:

```
https://KULLANICI_ADI.github.io/koraytasancom/
```
